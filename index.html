HTML

  • The !DOCTYPE is not case-sensitive because it is a declaration. Declarations are not case-sensitive; HTML elements are case sensitive
  • lang="en" tells the language for the webpage
  • HTML attributes are always defined in the beginning element, never the closing element
  • meta tag defines metadata about an HTML document, meta tags go inside the head element
  • definition and usage https://www.w3schools.com/tags/tag_meta.asp
  • The head element contains information about the webpage
  • the head element is a container for metadata, the user won't see info. in the head or meta elements
  • the head element in this case defines the following three elements charset, http-equiv, viewport
  • meta elements contain information about the page that is used by the browser
  • The Body element represents the visible content shown to the user.
